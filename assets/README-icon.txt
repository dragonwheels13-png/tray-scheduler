Place your icon files here:

  icon.ico       — Windows app icon (required for .exe build)
                   Recommended: 256x256px, ICO format
                   Free converter: https://convertio.co/png-ico/

  tray-icon.png  — System tray icon (16x16 or 32x32px PNG)
                   Should be a small, simple icon that looks good at 16px

If icon.ico is missing, electron-builder will use its default icon.
If tray-icon.png is missing, the app falls back to a built-in green dot.
